${GITHUB_REPOSITORY} Arch Linux Repository

This repository hosts a set of driver packages for the Alienware Steam Machine for Arch Linux. This can help getting SteamOS running.

Adding the repo to Arch Linux

Add the following lines to your pacman.conf: 

[steam-machine]
SigLevel = Optional TrustAll
Server = https://sharkwouter.github.io/steam-machine-driver-repo/
